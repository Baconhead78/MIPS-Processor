# MIPS-Processor
Implementation of a processor with the MIPS architecture and the HDL Verilog.

## Modules:
- [x] ALU.
- [x] ALU Control.
- [x] Register File.
- [x] Sign extend.
- [x] Shift left 2 (2 from 2).
- [x] Instruction Memory.
- [x] Program Counter.
- [x] Main Control Unit.
- [x] Data Memory.
- [x] MUX (4 from 5).
- [x] Adder(2 from 2).
